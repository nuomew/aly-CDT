# 阿里云流量查询系统 - 说明文档

## 一、项目概述

### 1.1 项目名称
阿里云CDT流量查询系统

### 1.2 项目描述
通过阿里云AccessKey查询云数据传输(CDT)流量使用情况的Web应用系统，包含安装向导、管理员后台和前端流量展示功能。

### 1.3 技术栈
- **后端**: PHP 7.2+ (兼容PHP 8.x)
- **数据库**: MySQL 5.6.50+
- **Web服务器**: Nginx
- **前端**: HTML5 + CSS3 + JavaScript (原生)
- **API**: 阿里云CDT API (2021-08-13版本)

## 六、项目总结

### 6.1 已完成功能

#### 安装向导 (`/install/`)
- 环境检测：检测PHP版本、扩展、目录权限
- 数据库配置：自动创建数据库和数据表
- 管理员设置：创建管理员账号
- 安装锁：防止重复安装

#### 管理员后台 (`/admin/`)
- 登录/退出：安全登录（已移除验证码）
- 控制台：统计概览、快捷操作
- 阿里云配置：添加/编辑/删除/测试AccessKey
- 流量统计：趋势图表、实例排行
- 系统设置：网站名称、缓存配置、自动刷新、流量提醒、邮件通知
- 操作日志：记录所有管理操作

#### 前端展示 (`/index.php`)
- 流量概览：今日/本月流量统计
- 流量进度条：显示本月流量使用百分比
- 趋势图表：最近7天流量变化
- 实例排行：TOP 10流量排行
- 自动刷新：可配置刷新间隔（不依赖定时任务）

#### 自动刷新功能
- 前端JavaScript定时刷新（不依赖服务器定时任务）
- 可配置刷新间隔（1-60分钟）
- 刷新后自动更新页面数据

#### 流量提醒功能
- 可配置最大流量限制（GB）
- 可配置提醒阈值（百分比）
- 流量达到阈值时自动发送邮件提醒
- 每天最多发送一次提醒邮件

#### 邮件通知功能
- 支持SMTP协议发送邮件
- 支持SSL/TLS加密
- 支持多个收件人（逗号分隔）
- 流量提醒邮件模板

#### API接口 (`/api/`)
- traffic.php：流量查询接口
- auto_refresh.php：自动刷新和提醒检查接口
  - refresh：刷新流量数据
  - check_alert：检查流量阈值
  - status：获取系统状态

### 6.2 核心类库

| 文件 | 功能 |
|------|------|
| Database.php | 数据库操作类(PDO) |
| AliyunCdt.php | 阿里云BSS API调用类（使用账单API获取流量） |
| Helper.php | 辅助函数类(加密、响应等) |

### 系统设置说明

#### 自动刷新设置
- **启用自动刷新**：开启后前端页面会自动定时刷新流量数据
- **刷新间隔**：建议设置5-10分钟，避免频繁请求

#### 流量提醒设置
- **最大流量(GB)**：设置每月最大流量限制，用于计算使用百分比
- **提醒阈值(%)**：当流量达到此百分比时发送邮件提醒

#### 邮件通知设置
- **SMTP服务器**：如 `smtp.qq.com`、`smtp.163.com`
- **端口**：SSL加密通常使用465端口
- **加密方式**：推荐使用SSL
- **收件人邮箱**：多个邮箱用逗号分隔

### 定时任务配置(Linux Crontab)
```bash
# 每5分钟刷新一次流量数据
*/5 * * * * curl -s "https://your-domain.com/admin/traffic.php?action=refresh" > /dev/null 2>&1
```

### Nginx配置示例
```nginx
server {
    listen 80;
    server_name your-domain.com;
    root /path/to/aliyun-traffic;
    index index.php;
    
    location / {
        try_files $uri $uri/ /index.php?$query_string;
    }
    
    location ~ \.php$ {
        fastcgi_pass unix:/var/run/php/php7.4-fpm.sock;
        fastcgi_index index.php;
        include fastcgi_params;
        fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name;
    }
    
    # 禁止访问敏感文件
    location ~ /\.(env|git|htaccess) {
        deny all;
    }
}
```

## 九、使用说明

### 安装步骤
1. 将项目上传到服务器
2. 复制 `.env.example` 为 `.env`
3. 配置Web服务器指向项目根目录
4. 访问 `http://your-domain.com/install/` 开始安装
5. 按向导完成环境检测、数据库配置、管理员设置
6. 安装完成后删除 `install` 目录

### 后台使用
1. 访问 `http://your-domain.com/admin/` 登录后台
2. 在"阿里云配置"中添加AccessKey
3. 点击"测试"验证配置是否正确
4. 在"流量统计"中点击"刷新数据"获取最新流量

### 前端访问
- 直接访问 `http://your-domain.com/` 即可查看流量数据
